# Pasta de Imagens

Coloque as imagens dos animais nesta pasta.

## Como usar:

1. Coloque as imagens nesta pasta: `src/main/resources/static/images/`
2. No cadastro do animal, use apenas o nome do arquivo (ex: `rex.jpg`)
3. A imagem será acessível em: `http://localhost:8080/images/nome-do-arquivo.jpg`

## Exemplo:

- Arquivo: `src/main/resources/static/images/rex.jpg`
- No campo foto do animal: `rex.jpg`
- URL completa: `http://localhost:8080/images/rex.jpg`
