# Imagens do Frontend

Esta pasta contém as capturas de tela do frontend para documentação.

## Arquivos necessários:

1. `login.png` - Tela de login do sistema
2. `animais-lista.png` - Listagem de animais em cards
3. `animais-cadastro.png` - Formulário de cadastro de animais
4. `monitoramento-iot.png` - Tela de monitoramento IoT
5. `estatisticas.png` - Dashboard de estatísticas

## Como adicionar as imagens:

1. Salve as capturas de tela nesta pasta com os nomes acima
2. Formatos aceitos: `.png`, `.jpg`, `.jpeg`
3. O README.md principal já está configurado para exibir essas imagens
